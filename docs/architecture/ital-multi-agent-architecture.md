# ITAL Multi-Agent Architecture

## Overview
ITAL is the Nutri Agent operating system for Victuals. It is organized as a hierarchical multi-agent workforce rather than a single chatbot-style assistant.

## Agent hierarchy
- Orchestrator Agent
- Human Nutrition Layer
  - Diagnostic Agent
  - Nutritional Specialist Agent
  - Disease and Clinical Agent
  - Biochemical Pathway Agent
- Product Intelligence Layer
  - Product Registry Agent
  - Product Composition Agent
  - Nutrient Translation Agent
- Commerce Layer
  - Grocery Agent
  - Budget Agent
  - Merchant Agent
- Plan Management Layer
  - Subscription Plan Agent
  - Token Utility Agent
  - Allocation Agent
- Economic Layer
  - Nutritional Economics Agent
  - Pricing Strategy Agent
  - Treasury Agent
- Population Nutrition Layer
  - Population Agent
  - Humanitarian Agent
- Learning Layer
  - User Feedback Agent
  - AI Critic Agent
  - Reinforcement Learning Agent
- Governance Layer
  - Nutrition Governance Agent
  - Compliance Agent

## Execution pattern
1. the orchestrator receives a request
2. it selects the relevant specialized agents
3. it aggregates outputs into a single explainable recommendation
4. it returns plan-aware, registry-aware, and commerce-aware recommendations

## Platform value
This structure enables Victuals to support individual users, market hubs, plan owners, governments, humanitarian programs, and nutrition commerce networks in a unified operating model.
