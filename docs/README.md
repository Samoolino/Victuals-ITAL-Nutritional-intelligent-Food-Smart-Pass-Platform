# Documentation Index

This folder contains the architectural, API, governance, and nutrition documentation for Victuals.

- `docs/architecture/` — enterprise architecture, ITAL multi-agent architecture, and platform design intent
- `docs/api/` — API conventions, service interfaces, and integration contracts
- `docs/governance/` — roles, policies, operating standards, and contribution expectations
- `docs/nutrition/` — nutrition ontology, evidence, and guidance models
- `docs/contracts/` — smart contract modules, event definitions, and integration notes
