# Applications

This directory contains the primary user-facing and operator-facing experiences for Victuals.

## Modules
- `apps/user-dapp/` — user access, nutrition planning, wallet, and redemption experience
- `apps/market-hub-portal/` — market owner and market hub operations
- `apps/plan-owner-portal/` — subscription plan design and funding oversight
- `apps/platform-admin/` — governance, compliance, and platform administration
- `apps/analytics-portal/` — reporting and impact dashboards
- `apps/registry-admin/` — registry administration and source coordination
