# User DApp

The user experience layer for Victuals provides users with a single place to manage nutrition access, wallet state, plan eligibility, product discovery, and redemption workflows.

## Responsibilities
- view nutrition entitlements and plan balances
- browse eligible products and nutrition recommendations
- authorize wallet-based redemption and settlement flows
- access reporting and plan progress insights

## Alignment
This module is the primary experience surface for users and is aligned with the ITAL nutrition planning and access services.
