# Services

This directory contains the application services that implement Victuals business workflows.

## Service catalog
- `services/user-access-service/` — entitlement, wallet, and access orchestration
- `services/plan-management-service/` — plans, allocation, and subscription administration
- `services/global-registry-service/` — product, merchant, and registry intelligence
- `services/nutrition-intelligence-service/` — ITAL agent orchestration and nutrition reasoning
- `services/marketplace-matching-service/` — product and merchant matching for commerce flows
- `services/merchant-network-service/` — merchant onboarding and merchant network operations
- `services/settlement-service/` — settlement, treasury, and payout processing
- `services/wallet-service/` — wallet lifecycle and device authorization integration
- `services/reporting-service/` — impact, program, merchant, and user reporting
- `services/compliance-service/` — policy and compliance evaluation
- `services/auth-service/` — identity and role-based access control
- `services/notification-service/` — alerts and notifications
- `services/payment-orchestrator/` — payment and conversion coordination
